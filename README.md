# 🎯 MOODSPOT
### 감정을 이해하는 AI 문화 큐레이터

> "오늘 너무 지쳤는데… 어디 가면 좋지?"  
> MOODSPOT이 감정을 읽고, 딱 맞는 하루를 설계해드립니다.

---

## 📋 프로젝트 소개

MOODSPOT은 **감정 기반 AI 문화 큐레이션 모바일 앱**입니다.  
감정 입력 → AI 분석 → 문화 처방 → 장소 추천 → 코스 자동 생성까지 끊김 없는 UX 흐름을 제공합니다.

기존 서비스는 인기/위치 기반 추천에 그쳤지만, MOODSPOT은 **지금 나의 감정 상태**를 이해하고 행동까지 연결합니다.

---

## ✨ 핵심 기능

| 기능 | 설명 |
|------|------|
| 💬 **AI 감정 대화** | 공감 기반 인터랙션으로 감정 진입 장벽 최소화 |
| 🎭 **AI 문화 처방** | 감정 → 미디어아트 / 독립서점 / 카페 등으로 변환 |
| 🗺️ **AI 산책 코스 생성** ⭐ | 전시→서점→카페→산책, 하루 흐름 자동 설계 |
| 📍 **지도 기반 탐색** | 코스 경로 시각화 + 필터 + 스와이프 시트 |
| 🧠 **AI 취향 학습** | 행동 데이터 기반 추천 정밀도 지속 향상 |
| 👥 **추천 코스 커뮤니티** | 인기 코스 공유 + 좋아요 + 후기 |

---

## 🔄 사용 흐름

```
감정 입력 → AI 대화 → 문화 처방 → 추천 장소 → 산책 코스 → 지도 → 추천 코스 → 취향 분석
```

## 🛠️ 기술 스택

**앱 (iOS / Android)**
- React Native + Expo
- TypeScript

**백엔드**
- Node.js + NestJS
- WebSocket (Socket.IO) — 실시간 AI 대화

**데이터베이스**
- PostgreSQL (메인 DB)
- Redis (캐시 / 세션)

**웹 / 관리자**
- Next.js + Vercel

**외부 API**
- 🧠 Claude API (Anthropic) — 감정 분석, 문화 처방, 코스 생성
- 🗺️ 카카오맵 API — 장소 검색, 경로 안내
- 🔔 Firebase FCM — 푸시 알림

**인프라**
- Vercel (웹/관리자), Railway (API 서버), AWS S3 (파일)
